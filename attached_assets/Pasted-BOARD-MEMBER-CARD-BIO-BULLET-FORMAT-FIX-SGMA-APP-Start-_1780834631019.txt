BOARD MEMBER CARD BIO BULLET FORMAT FIX — SGMA APP

Start this small UI fix now.

Scope:
Improve the Board of Directors member card bio display.

Current problem:
In /board/current, the board member bio appears as one long paragraph. It looks crowded and unprofessional on mobile, especially when the bio contains Arabic and German text.

Required:
Display each board member bio as a clean bullet list.

Do not change database schema.
Do not change backend unless absolutely necessary.
Do not change board permissions.
Do not change unrelated modules.

Current architecture:

* PostgreSQL + Drizzle
* JWT auth
* Arabic RTL
* Mobile-first
* No Firebase
* No MySQL migration

==================================================

======================

In the board member card, convert the bio text into bullet points.

Example stored bio:

أخصائي أمراض قلب من سوريا وألمانيا

عضو هيئة تأسيسية في SGMA

منسق فريق القلبية

• أخصائي أمراض قلب من سوريا وألمانيا
• Oberarzt für Kardiologie im Klinikum Itzehoe
• عضو هيئة تأسيسية في SGMA

==================================================
B. BIO PARSING RULES
====================

Create a small helper function on the frontend, for example:

formatBoardBioToBullets(bio: string): string[]

Rules:

1. Split bio by new lines.
2. Also support lines that already start with:

   * "-"
   * "•"
   * "."
3. Trim whitespace.
4. Remove empty lines.
5. Remove duplicated bullet characters from the beginning.
6. Return clean bullet text items.

If bio has no line breaks:

* Still display it as one bullet point instead of a paragraph.

Do not mutate stored database value.
Only change display formatting.

C. BOARD CARD UI
================

Update the board member card UI in:

/board/current

Bio display should be:

<ul>
  <li>...</li>
</ul>

Arabic RTL-friendly styling:

* list-inside or custom bullet
* comfortable spacing
* text-sm or readable size
* line-height relaxed
* bullet should not break RTL layout
* German text should still display correctly

Suggested styling:

* Use a vertical list.
* Each item has a small colored dot/bullet.
* Text color: muted foreground.
* Spacing between points: 4–6px.
* Do not use a dense paragraph.

If the bio is empty:
Show nothing or:
لا توجد نبذة متاحة
