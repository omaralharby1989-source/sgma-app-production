==========================

The /home page is the user’s “حسابي / My Account” page.

It must allow the logged-in user to:

1. View all account information.
2. Edit all editable information that the user entered during registration.
3. Change profile image/avatar.
4. Change password.
5. Logout.

The design must remain:

* Arabic RTL
* Mobile-first
* Clean
* Sectioned
* Not crowded
* Clear save/cancel behavior
