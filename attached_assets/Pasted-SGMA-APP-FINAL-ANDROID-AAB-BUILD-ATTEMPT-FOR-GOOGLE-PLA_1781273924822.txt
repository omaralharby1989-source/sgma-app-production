C. BUILD STEPS TO ATTEMPT
=========================

First run:

pnpm --filter @workspace/sgma-app2 run typecheck

Then run:

VITE_API_BASE_URL=https://sgma-app.org VITE_ADMOB_TEST_MODE=true pnpm --filter @workspace/sgma-app2 run build:cap

Then run:

pnpm --filter @workspace/sgma-app2 exec cap sync android

Then try:

cd artifacts/sgma-app2/android
./gradlew bundleRelease

If gradle wrapper is missing, check whether the Android project has the correct Gradle files.
If Android SDK/JDK/Gradle is unavailable in Replit, do not fake success. Stop and report the exact blocker.
